OpenTraining

单车训练很简单，也很难。每次在写计划的时候，我在想能不能用编程语言实现这些简单的计算。

这个仓库的初衷就是这样：希望能实现自动生成一个训练周期的训练日程，并且能够输出在 A4 纸上，简单剪裁之后，就可以指导每一天的训练内容。
